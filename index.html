April 7, 2020: Valorant Closed Beta

Valorants closed beta launched and was availible through twitch drops. The game was in a very early stage compared to now, only half the agents and maps and graphics were very different.

twich drops are a way of rewarding the community through viewtime with rewards
At this point in release Valorant was in a very early stage, where graphics, agents and abilities, all the way to skins were very different.
During the beta skins were open to purchase but they were not going to be availible on the full release of the game.
The games UI was also very different and ranking system was down a rank compared to now with the highest rank being named "valorant".

  • Viper - agent 2
  • Omen - agent 3

June 2, 2020: Valorant Full Release // Episode 1

valorant fully released to the public for anyone to play from their website. This was a huge update for valorant with many balancing changes and minor tweaks.

In update 1.0 Reyna, agent - 11, was added to valorant. Many agent balancing changes were made, increasing durations, cooldowns, ultimate ability points required etc.
There was a new map introduced aswell, Ascent, this maps gimick was that it had a very large "mid" to fight for control
Many minor map changes were made along with some minor QOL changes.
For full patch notes on update 1.0 click here.

In update 1.05 valorant added the new mode "deathmatch" which was a 1 Vs All FFA gamemode primarily focused on aim.
Aswell the act changed to act 2 adding a new battlepass with new skins to progress through.
Killjoy, agent - 4, was also added this update.
The competitive gamemode finally went live and changed the top rank to be the top 500 players called "Radiant".
For full patch notes on update 1.05 click here.

Widely considered the act with the best knife skin in a battle pass update 1.11 changed the game to act 3.
Skye, agent - 14, was added
The map icebox entered competitive rotation.
Overall a fairly minor update with just a new agent and many agent balancing changes.
For full patch notes on update 1.11 click here.
